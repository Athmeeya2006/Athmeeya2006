### Projects

**[Erdos-Renyi Random Graph Evaluation](https://github.com/Athmeeya2006/erdos-graph)**
Confirmed the s^(-3/2) cluster-size power law at the percolation threshold via 1,000+ Monte Carlo trials on graphs up to 10,000 vertices. Benchmarked O(n^2) naive generation against Batagelj-Brandes linear-time algorithm.
`Python` `NetworkX` `SciPy` `Monte Carlo`
